# 🎬 Netflix UI Assessment

A Netflix-style desktop web application built using React.js. This project was developed as part of an assessment and demonstrates UI design, performance optimization, and interactive features.

---

## 🚀 Live Demo
👉 https://lustrous-dasik-d90f7c.netlify.app/

---

## 🛠️ Tech Stack
- React.js
- JavaScript (ES6)
- Axios
- CSS (Custom Styling)

---

## ✨ Features

### 🎥 Core Features
- Movie browsing UI (Netflix-style layout)
- Dynamic data fetching using API
- Responsive grid layout

### 🔍 Search Functionality
- Search by title, year, or ID
- Debounced search to optimize performance

### ⚡ Performance Optimization
- Lazy loading (infinite scroll)
- Prevention of unnecessary re-renders
- Memory leak handling using cleanup functions

### 🎬 Interactive UI
- Movie preview modal on click
- Smooth UI transitions and hover effects

### 📌 User Features
- Watchlist (stored in localStorage)
- Watch history tracking

### 🌐 Network Handling
- Offline/Online detection
- UI updates based on network status
